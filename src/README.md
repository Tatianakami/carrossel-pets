## ✨ Funcionalidades
- 🖼️ Carrossel de imagens responsivo
- 📲 Navegação por botões e indicadores
- 🗂️ Upload de novas imagens (JPEG/PNG/WebP)
- 💾 Persistência no localStorage
- 📱 Compatível com mobile
